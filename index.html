The Greenhorn Gaming Engine

A simple, easy to use HTML5 canvas based gaming engine written in CoffeeScript by Seth David Bullock.

Download as .zip Download as .tar.gz View on GitHub

History and Inspiration

This project is inspired and draws heavily from Andy Harris' simpleGame.js. Andy was my introductory Computer Science professor at IUPUI. Over the course of a semester I had an opportunity to work with Andy along with some of my fellow students using his simpleGame.js engine. I have played video games all my life, but never had any experience writing them. It was so much fun that I bought his book and worked all the way through it over the course of just a few weeks. I am the kind of person who has to do something by hand in order to truly understand it. Which is why when I discovered CoffeeScript this past October I decided that the best way to learn it and at the same time get to write some video games was to take simpleGame.js completely apart and put it all back together again in this new language.

Using the Engine

If you would like to use the Greenhorn Gaming Engine to create your own games, all you must do is download the file lib/Greenhorn.js and include it in your webpage like this:

<script type="text/javascript" src="path/to/Greenhorn.js"></script>

Some rough documentation can be found here. Please forgive it's incompleteness, just like this library, it's still a work-in-progress.
